# /marketing — base de materiais do Leadership MCP

Materiais reutilizáveis de adoção do projeto. Tudo aqui é **público e sanitizado**:
mensagens, templates e guias genéricos. Estratégia sensível (rede de facilitadores,
listas de alvos, credenciais, decisões comerciais) **não vive aqui** — fica no board
privado do projeto (Linear).

## Estrutura

- [`parcerias/`](parcerias/) — canal primário **B2B2C**: materiais para empresas de
  treinamento de liderança e coaches revenderem/entregarem o produto (white-label via fork).

> Próximas pastas planejadas (ver Linear STR-204): `lancamento/` (kit repetível),
> `semeadura/` (kits por comunidade), `identidade/` (guia de marca), `demo/` (roteiro do GIF).

## Como usar

Copie o material, adapte o que estiver `[entre colchetes]` e envie. Cada arquivo diz
para quem serve e como usar no topo. Nada aqui exige código.
